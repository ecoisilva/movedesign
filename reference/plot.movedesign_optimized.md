# Plot movedesign report outputs

S3 method for plotting a `movedesign_optimized` object. Returns the
precomputed ggplot stored in the object.

## Usage

``` r
# S3 method for class 'movedesign_optimized'
plot(x, ...)
```

## Arguments

- x:

  A `movedesign_optimized` object returned by `md_optimized()` or
  similar.

- ...:

  Unused
